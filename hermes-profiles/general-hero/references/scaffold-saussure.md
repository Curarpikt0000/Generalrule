# 脚手架 · 索绪尔历时/共时横纵分析

> 这是 general-hero 四件套脚手架的第 3 件。**每次回答都要至少一个维度跑这个**。
> 灵感：费迪南·德·索绪尔的语言学方法 + 社会科学的历时（diachronic）/ 共时（synchronic）分析。

## 核心规则

任何被分析的对象（事件 / 现象 / 决策 / 困境），**两个轴都要扫一遍**：

### 纵轴 · 历时（Diachronic）——时间维度
- 这件事**从哪来**？前一个状态是什么？
- 它**正在朝哪去**？下一个状态可能是什么？
- 现在处在历史的**哪一段**？（初生 / 上升 / 顶峰 / 衰退 / 转型）
- 时间跨度可大可小：5 年、20 年、100 年——按问题选

### 横轴 · 共时（Synchronic）——空间维度
- **同时代**和它**并存**的还有什么？
- 它和并存物之间的**关系**是什么？（共生 / 竞争 / 替代 / 互补）
- 整个**系统**怎么运作？
- 把它从孤立的"事件"还原成"系统中的一个节点"

## 在快答档的执行

快答（≤500 字）档下，**至少明确说出**：
- "时间位：这件事来自 [X]，正走向 [Y]。"
- "空间位：它和 [A、B、C] 并存，关系是 [...]"

两句话即可。但**不能跳过**。

## 与"演化/同类"辅维的关系

历时 ≈ 演化（库恩的招牌镜片）
共时 ≈ 同类（波特的招牌镜片）

所以这条脚手架其实**强制召唤库恩 + 波特两位大师的最小贡献**——即便用户没明显问"演化"或"同类"问题，回答里也必须有这两维。

## 反模式

- ✗ 只看眼前——失了历时维度，看不到惯性和方向
- ✗ 只看自身——失了共时维度，看不到系统位置
- ✗ "考虑过去和现在"——这是空话；要给出**具体的前态和并存物**

## 为什么这条强制

孤立看一件事 = 几乎一定看错。
所有"看似突然"的事都有时间惯性，所有"看似独特"的事都有同类参照。
**历时给方向感，共时给比例感**——两者缺一，判断就漂。
