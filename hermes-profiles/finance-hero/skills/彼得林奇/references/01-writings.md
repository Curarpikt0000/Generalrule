## 二、反复出现的核心论点（真信念）

### 2.1 身边股主义（Invest in What You Know）

**出现频率：极高，贯穿两本书**

> "决不投资于任何你画不出来的东西。"

**核心含义**：
- 最大的投资机会就在你日常生活里
- 普通人比专业投资者更有优势——你每天都在使用产品
- 医生知道哪家药厂好，银行经理知道哪家银行强

**原文摘录**：
> "如果你在日常生活中注意观察，就会发现那些成功的大公司。问题是大多数人视而不见。"

---

### 2.2 买股票就是买企业一部分

**出现频率：高**

> "当你投资债券，你只是放了一笔贷款。但如果你投资股票，你就买下了公司的一部分。如果公司生意兴旺，你可以分享公司的收益。"

**核心含义**：
- 股票是企业所有权凭证，不是赌博筹码
- 长期持有优秀公司，收益远超债券
- 股价最终由公司收益决定

---

### 2.3 时间是普通投资者最大的优势

**出现频率：极高**

**核心含义**：
- 年轻时有时间，年老时有资本
- 越早开始越好，定期投资不看市场涨跌
- 每次下跌10%后追加1000美元，83000美元变成6295000美元

**数据支撑**：
- 1940年投资1000美元标普500，52年后变成333793美元
- 每月定投1000美元，52年后变成3554227美元

---

### 2.4 股价最终由收益决定

**出现频率：高**

> "如果收益增加，股价注定要涨，可能不会马上就涨，但终究会涨。"

**核心含义**：
- 关注公司季度/年度报告中的销售额、收益、订单变化
- 不要关注每日股价波动
- 短期股价可能偏离基本面很久

---

### 2.5 公司六分法

**出现频率：中等（林奇独创）**

| 类型 | 特征 | 策略 |
|------|------|------|
| 稳定增长型 | 低增长，如宝洁、可口可乐 | 长期持有 |
| 缓慢增长型 | 高分红，如电力公司 | 持有收息 |
| 周期型 | 汽车、钢铁 | 低市盈率卖出，高市盈率买入 |
| 资产型 | 拥有土地、品牌 | 关注隐藏资产 |
| 快速增长型 | 小盘零售、科技 | 快速增长后警惕减速 |
| 困境反转型 | 破产边缘 | 高风险高回报 |

## 四、林奇13条投资准则

| 准则 | 原文 |
|------|------|
| 第一准则 | 决不投资于任何你画不出来的东西 |
| 第二准则 | 你不能从后视镜里看到未来 |
| 第三准则 | 只要你挑选基金，很可能就选中一只好基金 |
| 第四准则 | 办公室奢华程度与股东分红递减幅度成正比 |
| 第五准则 | 债券收益高于股市分红6%时卖股买债 |
| 第六准则 | 不是所有的普通股都同样普通 |
| 第七准则 | 股价大跌对心安理得的持股人是最好的药方 |
| 第八准则 | 如果你喜欢这家商店，多半会爱上它的股票 |
| 第九准则 | 内部人员买自己公司股票是利好讯号 |
| 第十准则 | 对经营而言，竞争不如垄断 |
| 第十一准则 | 投资于年度报告中彩照最少的公司 |
| 第十二准则 | 当分析员感到厌烦之时，正是买进之机 |
| 第十三准则 | 绝不掏钱作悲观主义者 |

## 五、经典投资案例

### 克莱斯勒
- 从破产边缘到小型货车成功
- 林奇持续加仓至5%上限
- 结果：利润超1亿美元，股价翻50倍

### 沃尔沃
- 股价低于净现金（34美元=每股现金）
- 华尔街忽视，被认为"乏味"
- 结果：18个月翻4倍，赚7900万美元

### 耐克
- 1987年连续20个季度收益增长
- 股价从7美元涨到90美元（12倍）
- 关键：订单增加是领先指标

### 强生
- 连续10年提高红利，收益翻4倍
- 股价被低估后回归
- 关键：回购股票提高每股收益
