## 一、重大决策案例

### 1. 1992年做空英镑（"黑色星期三"）

| 维度 | 内容 |
|------|------|
| **背景** | 英国加入欧洲汇率机制(ERM)，汇率被高估；索罗斯发现英国政府无法长期维持高汇率 |
| **决策** | 做空英镑，投入约100亿美元 |
| **逻辑** | "我看到了不可持续性"——英国政府要么提高利率（损害经济），要么被迫贬值 |
| **结果** | 盈利约10亿美元，被称为"打破英格兰银行的人" |
| **事后反思** | 这是他最成功的交易，但也引发政治争议（被指责为"投机者"） |

**来源**：Investopedia, NPR Planet Money, Charlie Rose 1995采访

### 2. 1997-1998亚洲金融危机

| 维度 | 内容 |
|------|------|
| **背景** | 东南亚货币被高估，银行体系脆弱 |
| **决策** | 量子基金做空泰铢等东南亚货币 |
| **逻辑** | 反射性：货币贬值→银行资产负债表恶化→进一步贬值 |
| **争议** | 被东南亚国家指责为危机的制造者 |
| **事后反思** | 索罗斯否认"制造危机"，称只是识别了不可持续的局面 |

**来源**：The New Yorker, Bloomberg

## 二、决策模式总结

### 索罗斯式决策特征

1. **识别不可持续性**：寻找"这个逻辑无法延续"的临界点
2. **利用反射性**：预期会导致进一步发展（繁荣或崩溃）
3. **不对称风险**：亏损有限（"最多损失几百万"），盈利无限
4. **政治意识**：清楚自己的交易可能产生政治后果

## 三、矛盾记录

### 矛盾1：投机者 vs 市场效率促进者
- 索罗斯批判市场原教旨主义
- 但他的交易本身加剧了市场波动
- 他曾说："我赚钱的方式证明了市场不完美"

### 矛盾2：慈善 vs 政治干预
- 开放社会基金会自称促进民主
- 被匈牙利、俄罗斯等国政府定义为"外国势力干预"
- 欧尔班称其建立"黑手党政权"
