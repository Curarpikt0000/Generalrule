# 本杰明·格雷厄姆重大投资决策案例研究

> 整理自《聪明的投资者》"本杰明·格雷厄姆生平简介"及后记部分

---

## 案例一：1919年 Savold 轮胎公司

### 背景
- **时间**：1919年4月
- **市场环境**：第一次世界大战后，汽车企业正处于被市场疯狂追捧的时代，新兴交通行业概念炙手可热
- **格雷厄姆年龄**：25岁（1914年毕业后进入华尔街）

### 格雷厄姆的决策逻辑
- 跟随市场热潮，在该股票上市的第一天大量买入
- 反映了当时市场对汽车题材的非理性追捧心态

### 结果
- 当天获利：**250%**（一天之内收益翻2.5倍）
- 同年10月，公司爆出**欺诈丑闻**
- 股票变得**一文不值**

### 教训
1. **追逐热门题材的风险**：即使在上市首日获得惊人回报，也可能是市场非理性的体现
2. **欺诈风险**：没有基本面支撑的热门概念，一旦曝光即归零
3. **短期暴利不可持续**：单纯依赖市场情绪而非企业价值，风险极高
4. **这也是格雷厄姆从投机转向价值投资的重要诱因之一**

## 案例二：1925年 北部管道公司（Northern Pipeline）

### 背景
- **时间**：1925年
- **契机**：与美国州际商业委员会合作，对石油管道公司晦涩难懂的年报进行深入研究
- **格雷厄姆状态**：已成为专业的证券分析师，专注个股微观研究

### 格雷厄姆的决策逻辑
1. **深入基本面研究**：对年报进行细致分析，发现账面上看不到的价值
2. **发现价值被低估**：
   - 公司拥有至少价值**每股80美元**的高等级债券
   - 而当时股价仅为**65美元**
   - 存在约23%的价值折价
3. **积极股东行动**：买入后，敦促公司管理层提高分红率
4. **估值回归逻辑**：当市场价格低于内在价值时买入，等待价值被发现

### 结果
- 持有**3年后**，以**110美元**卖出
- 获利幅度：约69%（65→110）
- 这一案例奠定了格雷厄姆作为"对股票进行微观研究大师"的声誉

### 教训
1. **深度研究创造优势**：当市场忽视或无法理解某类资产时，深入研究者可以获得超额回报
2. **安全边际原则**：以低于内在价值的价格买入，为错误预留空间
3. **主动与被动结合**：发现价值后，积极推动管理层释放价值
4. **耐心等待**：价值实现需要时间，3年的持有期在价值投资中属于正常
5. **专业化研究的重要性**：与监管机构合作获取的信息优势

## 案例三：1929-1932年 大萧条

### 背景
- **时间**：1929年9月 - 1932年
- **事件**：美国股市崩盘，随后进入史上最严重的大萧条
- **市场状况**：道琼斯指数从1929年高点下跌约90%，无数投资者破产
- **格雷厄姆年龄**：约35岁，已建立自己的合伙投资机构

### 格雷厄姆的决策与结果
- 在大崩盘前显然持有大量股票仓位
- **亏损接近70%**
- 这一打击几乎摧毁了他的职业生涯

### 后续恢复
- 格雷厄姆"闯过了这一关"，并未被灾难击垮
- 随后"卷土重来，在大牛市的废墟上收获了大量有利的交易"
- 1936-1956年退休期间，年均收益率**14.7%**，高于同期市场整体**12.2%**的收益率

### 教训
1. **没有人能幸免于熊市**：即使是最优秀的投资者，在极端市场条件下也会遭受重大损失
2. **亏损的痛苦远大于盈利的快乐**：70%的亏损需要未来上涨233%才能回本
3. **活着最重要**：能够存活下来并恢复，是长期投资成功的关键
4. **熊市也是机会**：在大崩溃后的废墟上，反而能发现廉价优质资产
5. **杠杆的风险**：大萧条前很多投资者使用保证金交易，崩盘时被迫平仓
6. **分散化的必要性**：格雷厄姆没有破产，部分原因可能是他的组合虽然下跌但未归零

## 案例四：GEICO 政府雇员保险公司

### 背景
- **时间**：约1948年（后记描述的事件）
- **公司状况**：一家成长型企业，当时不被华尔街看好
- **市场环境**：因某方面原因，这笔交易被好几家重要机构拒绝
- **格雷厄姆身份**：格雷厄姆-纽曼公司合伙人

### 格雷厄姆的决策逻辑
1. **逆向投资**：选择被市场冷落的公司，避开热门高估的标的
2. **看重价值而非人气**：
   - 价格相对于当期利润不高等因素使得这笔交易显得有吸引力
   - 公司的业务具有某种"离不开"的性质——刚性的市场需求
3. **集中投资**：以大约**五分之一（约20%）的资金**开展并购
4. **利益绑定**：格雷厄姆-纽曼公司的利益与这笔新业务密切相关
5. **风险控制意识**：在后记点评中提到，格雷厄姆"尽力确保了他一旦清理GEICO股份时不会亏损"——说明他在买入时已经考虑了保本

### 结果
- 股价上涨到初始购买时的**200多倍**
- 单这一笔投资决策获取的利润，**大大超过20年内在合伙专业领域里广泛开展各种业务所获得的其他所有利润的总和**
- 许多持有该股票的基金参与者都成为了百万富翁

### 教训
1. **集中投资的机会**：当确定性高时，可以适当集中持仓，但需要充分的研究支持
2. **逆向投资的回报**：被市场冷落的优质公司，可能蕴含巨大机会
3. **能力圈原则**：格雷厄姆能够识别GEICO的价值，源于他对保险行业的深刻理解
4. **关键决策的重要性**：一次正确的重大决策，可能超越一辈子无数次普通决策的总和
5. **勇气与风险的平衡**：格雷厄姆有勇气投入20%的资金，是因为他做了充分的分析并控制了风险
6. **"离不开的业务"**：具有刚性需求、难以被替代的商业模式，是优质投资标的的特征
7. **长期持有**：股价200倍的上涨，需要时间来发酵

## 四大案例综合分析

### 决策风格的演变轨迹

| 阶段 | 时间 | 案例 | 风格 |
|------|------|------|------|
| 早期 | 1919 | Savold轮胎 | 跟随热点、投机获利 |
| 中期 | 1925 | 北部管道 | 价值发现、系统研究 |
| 挫折 | 1929-32 | 大萧条 | 重仓、杠杆的代价 |
| 成熟 | 1948+ | GEICO | 逆向集中、稳健获利 |

### 核心原则的形成

1. **从投机到投资**：格雷厄姆从Savold的教训中认识到，追捧热门是危险的
2. **从分散到集中**：早期分散化投资，成熟期敢于在确定性强的情况下集中持仓
3. **从亏损到风控**：大萧条的惨痛教训使他将"安全性"原则置于首位
4. **从被动到主动**：不仅发现价值，还推动价值实现

### 成功投资者的共同特质

- **研究深度**：对基本面深入研究，创造信息优势
- **心理素质**：面对极端市场（暴涨暴跌）保持理性
- **错误纠正**：从失败中学习，不断完善投资体系
- **勇气与纪律**：在别人恐惧时买入，在别人贪婪时卖出
- **长期视角**：不被短期波动左右，追求长期稳定回报
