# 本杰明·格雷厄姆完整人生时间线

> **来源说明**：本时间线综合自《聪明的投资者》书中"本杰明·格雷厄姆生平简介"（贾森·兹威格著）及维基百科英文版等权威资料。

---

## 早年岁月（1894-1914）

| 时间 | 事件 |
|------|------|
| 1894年5月9日 | 出生于伦敦，原名本杰明·格罗斯鲍（Benjamin Grossbaum），父母为英国犹太人 |
| 1895年（约1岁时） | 随家人迁居纽约 |
| 1903年（9岁） | 父亲去世，家庭陷入贫困。父亲曾是瓷器餐具和小塑像经销商 |
| 1907年（13岁） | 母亲借钱进行股票保证金交易，股灾中本金化为乌有。格雷厄姆永远记住了这段屈辱的经历 |
| 1914年（20岁） | 以全班第二名成绩毕业于哥伦比亚大学。学校三个系（英语、哲学、数学）邀请他留校任教，他选择进入华尔街养家 |

---

## 华尔街初露锋芒（1914-1929）

| 时间 | 事件 |
|------|------|
| 1914年 | 加入一家债券交易公司担任文员，随后晋升为分析师、合伙人 |
| 1919年4月 | 在Savold轮胎公司上市首日大赚250%，该公司同年10月爆出欺诈丑闻并变得一文不值 |
| 1925年 | 与美国州际商业委员会合作，深入研究北部管道公司年报，发现其债券价值远超股价（每股80美元债券，股价仅65美元）。买入后推动管理层提高分红，3年后以110美元卖出 |
| 1920年代 | 经营自己的合伙投资机构 |

---

## 大萧条与东山再起（1929-1936）

| 时间 | 事件 |
|------|------|
| 1929-1932年 | 大萧条期间亏损近70%，但挺过难关，在牛市废墟上收获大量有利交易 |
| 1934年 | 与戴维·多德合著《证券分析》出版，奠定"安全边际"和价值投资理论基础 |

---

## 格雷厄姆-纽曼时代（1936-1956）

| 时间 | 事件 |
|------|------|
| 1936年 | 与杰罗姆·纽曼（Jerome Newman）共同创立格雷厄姆-纽曼公司（Graham-Newman Corp.） |
| 1948年 | 格雷厄姆-纽曼以71.25万美元购入GEICO 50%股份。后因SEC监管要求，将GEICO股票分配给基金投资者。若持有1948年的100股（价值11,413美元）分到的GEICO股票至1972年，价值可达166万美元 |
| 1936-1956年 | 格雷厄姆-纽曼年收益率约14.7%，同期市场整体收益率为12.2% |
| 1949年 | 《聪明的投资者》出版。沃伦·巴菲特称其为"有史以来最棒的投资书" |
| 1956年 | 从格雷厄姆-纽曼公司退休 |

---

## 执教生涯与晚年（1956-1976）

| 时间 | 事件 |
|------|------|
| 1956年后 | 长期在哥伦比亚大学商学院任教，巴菲特为其学生之一 |
| 晚年 | 还在加州大学洛杉矶分校（UCLA）安德森管理学院授课 |
| 1976年9月21日 | 在法国普罗旺斯地区艾克斯（Aix-en-Provence）去世，享年82岁 |

## 格雷厄姆生命最后几年："傻事、有创造性的事、慷慨的事"

巴菲特在1976年《金融分析师杂志》上撰文纪念格雷厄姆时，提到格雷厄姆常说的一句话：

> "我希望每天都能做一件傻事、一件有创造性的事、一件慷慨的事。"

巴菲特评价道：**"在这些事情上，他最擅长的是最后一件——慷慨。"**

格雷厄姆的学生、 disciple 遍布华尔街，包括：
- 沃伦·巴菲特（Warren Buffett）——称格雷厄姆为其生命中第二重要的人（仅次于父亲）
- 欧文·卡恩（Irving Kahn）
- 沃尔特·施洛斯（Walter Schloss）
- 查尔斯·布兰德（Charles Brandes）
- 威廉·鲁安（William J. Ruane）
- 塞思·卡拉曼（Seth Klarman）等

## 身后影响

格雷厄姆被誉为"价值投资之父"，他的投资哲学强调：
- **独立思考**：不被群体情绪左右
- **安全边际**：永远不要支付过高的价格
- **内在价值**：股票是企业所有权的凭证，价格终将反映价值
- **Mr. Market寓言**：利用市场先生的情绪波动而非被其左右

巴菲特、芒格等价值投资者均承认格雷厄姆是他们的思想启蒙者。尽管格雷厄姆的方法在当代市场被认为"必要但不充分"（因对成长性强调不足），但他的核心原则至今仍是投资界的基石。
