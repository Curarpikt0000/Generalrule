# 格雷厄姆对话与访谈风格研究

## 一、格雷厄姆与巴菲特的师生关系

### 1.1 相遇与师徒情谊

**来源**: 《聪明的投资者》巴菲特序言和附录1
**可信度**: 高（直接引自巴菲特亲笔文章）

巴菲特在1950年19岁时阅读了《聪明的投资者》第一版，毅然决定前往哥伦比亚大学求学。他在1976年发表的纪念短文中写道：

> "要想在一生中获得投资的成功，并不需要顶级的智商、超凡的商业头脑或秘密的信息，而是需要一个稳妥的知识体系作为决策的基础，并且有能力控制自己的情绪，使其不会对这种体系造成侵蚀。"

**关键引用**（来源：《金融分析师杂志》，1976年11/12月刊）：
> "本杰明·格雷厄姆对我一生的影响，仅次于我的父亲。"

### 1.2 格雷厄姆拒绝巴菲特

**来源**: 巴菲特1984年哥伦比亚大学座谈会发言（《聪明的投资者》附录1）
**可信度**: 高（巴菲特亲述）

巴菲特在发言中提到：
> "在修完格雷厄姆的课程后，我提出到格雷厄姆一纽曼公司做义工，但他却以自我视过高为由拒绝了我；他是很看重这一点的。在经过多次请求后，他终于雇用了我。"

这展现了格雷厄姆的**谦逊原则**——即使面对热情的学生，也会保持警惕。

### 1.3 "80%格雷厄姆血液"

**来源**: 《聪明的投资者》译者序
**可信度**: 高（广泛引用的经典表述）

巴菲特曾说：*"在我的血管里，百分之八十流淌的是格雷厄姆的血液。"*

## 二、格雷厄姆的教学风格

### 2.1 哥伦比亚大学的课堂

**来源**: 巴菲特1984年发言
**可信度**: 高

巴菲特描述格雷厄姆的教学特点：
- **思想温和，不带说教或傲慢**
- **表达方式无疑是温和的**
- **思维广泛，对新知识一如既往地着迷**
- **能够把知识重新应用于表面上不相关联的问题**

### 2.2 "傻事、有创造性的事和慷慨的事"

**来源**: 巴菲特1976年纪念文《怀念本杰明·格雷厄姆》
**可信度**: 高（第一手回忆）

> "几年以前，在即将80岁之际，格雷厄姆向一位朋友表达了他的想法：希望每天都做一些'傻事、有创造性的事和慷慨的事'。"

巴菲特对此的解读：
> "他的第一个奇怪目标反映了这样一个事实：他善于使自己的想法不会带有任何说教或傲慢成分。尽管他的想法是强有力的，但是，它们的表达方式无疑是温和的。"

### 2.3 对学生的慷慨

**来源**: 巴菲特1976年纪念文
**可信度**: 高

> "他的第三项责任（慷慨），正是他比所有其他人都做得更成功的地方。我是以一个学生、一个雇员和一个朋友的身份认识格雷厄姆的。无论从哪一种关系来看（在他所有的学生、雇员和朋友看来），格雷厄姆在自己的想法、时间和精力等方面都表现出了毫无保留的慷慨。如果想寻求一种明晰的思维，那么没有比格雷厄姆更好的人选了。而且，如果需要获得鼓励和忠告，就可以随时去找格雷厄姆。"

## 三、格雷厄姆被追问时的即兴反应

### 3.1 对"掷硬币比赛"的回应

**来源**: 巴菲特1984年哥伦比亚大学发言
**可信度**: 高

巴菲特在座谈会上用一个精妙的**"掷硬币比赛"比喻**来回应有效市场理论家的质疑：

> "让我们设想一场全国性的掷硬币比赛...全美2.25亿居民都拿出1美元做赌注...经过10个早晨的10轮比赛，全美大约会剩下22万名连续10场获胜的赢家..."

然后他话锋一转：
> "如果这2.25亿只猩猩进行同样的比赛，结果也会与此完全相同——其中215只傲慢的猩猩将连续20次押对结果。但是，接下来，一些商学院的教授会毫不客气地指出如下事实..."

这个比喻展现了格雷厄姆学派面对学术质疑时的**幽默与智慧**——用类比揭示学术逻辑的荒谬。

### 3.2 对"40美分买1美元"的即兴领悟

**来源**: 巴菲特1984年发言
**可信度**: 高

巴菲特描述价值投资理念的传播方式：
> "附带说明一下，让我感到非常奇怪的是，人们要么会瞬间接受以40美分买进1美元的东西这一理念，要么永远也不会接受这一理念。这就像向某人灌输某种思想一样，如果这一理念不能立即俘获他，即使你再跟他说上几年，拿出历史纪录给他看，也无济于事。他们就是不能掌握这一概念，尽管它是如此简单明了。像里克·格林这种人，虽然没有受过正式的商业教育，却能马上理解这种价值投资法，并在5分钟后将其应用于自己的实战。我从来没见过任何人是在10年间逐渐接受这一理念的。"

这反映了格雷厄姆教学中一个特点：**理念的传授是顿悟式的，而非渐进式的**。

### 3.3 关于"安全边际"的桥比喻

**来源**: 巴菲特1984年发言
**可信度**: 高

> "你必须有能力对企业的内在价值有一个大体的估计，但是你并不能够给出准确的答案。这就是格雷厄姆所说要有一定安全性的意思。你不要用8000万美元去买进价值8300万美元的资产。你要使自己具有很大的安全性。在建筑一座桥梁时，尽管通过这座桥的卡车的重量通常只有10000磅，你也必须坚持其承重要达到30000磅。同样的原则也适用于投资。"

这个比喻展现了格雷厄姆教学的**直观性和实用性**——用日常生活中的工程学原理来说明投资哲学。

## 四、格雷厄姆与同行的对话

### 4.1 与戴维·多德的合作

**来源**: 巴菲特1984年发言
**可信度**: 高

格雷厄姆与戴维·多德合著《证券分析》，巴菲特指出：
> "格雷厄姆和多德致力于'以显著的安全性标准寻找价值与价格差异'的证券分析法"

多德在哥伦比亚大学举办投资夜校讲座，汤姆·纳普（后来的Tweedy-Browne创始人之一）正是听了他的讲座后被吸引，最终获得MBA学位并成为格雷厄姆的学生。

### 4.2 与费雪尔的比较

**来源**: 巴菲特1984年发言
**可信度**: 高

巴菲特在发言中虽未直接讨论费雪尔，但对比了两种投资风格：
> "我先来回顾一下曾于1954~1956年在格雷厄姆一纽曼公司工作过的四个人的经历...他们并不是根据首领所给出的信号来猜测掷硬币的结果的。他们的家长只是在如何猜测掷硬币的结果方面建立了一套理论，但每个学生是根据自己的方式来对理论加以应用的。"

这说明格雷厄姆的教学**鼓励独立思考**，学生们各有所得。

## 五、格雷厄姆的晚年与幽默

### 5.1 晚年的三个目标

**来源**: 巴菲特1976年纪念文
**可信度**: 高（第一手回忆）

格雷厄姆在80岁时表达的人生目标：
> "希望每天都做一些'傻事、有创造性的事和慷慨的事'。"

这三个目标的解读：
- **傻事** → 不自以为是，保持谦逊
- **有创造性的事** → 持续创新，不固步自封
- **慷慨的事** → 对学生、同事、朋友毫无保留

### 5.2 巴菲特关于格雷厄姆智识范围的评论

**来源**: 巴菲特1976年纪念文
**可信度**: 高

> "关于格雷厄姆在自己的专业领域所拥有的支配地位，最突出的一个方面在于，这并不是从全神贯注地关注某一个目标的狭隘思维活动中所取得的。相反，他的这种地位是几乎无法定义的广泛的智力活动所带来的副产品。毫无疑问，我从未遇到过思维如此广泛的人。惊人的记忆力，对新知识一如既往的着迷，以及能够把这些知识重新应用于表面上不相关联的问题，所有这一切使得他的思维方式在任何领域都会受到人们的喜爱。"

### 5.3 幽默引用

**来源**: 《聪明的投资者》正文
**可信度**: 高

格雷厄姆在书中引用威尔·罗杰斯（Will Rogers）的幽默：
> "不要打赌。将所有的积蓄用来购买一些优质股，并持有它们。等到价格上涨后将其卖掉。如果它不上涨，就不去买它。"

## 六、格雷厄姆的核心特质总结

根据巴菲特等人的回忆，格雷厄姆的对话和教学风格具有以下特征：

| 特质 | 具体表现 |
|------|----------|
| **温和** | 想法有力，但表达方式温和，不带说教或傲慢 |
| **慷慨** | 在想法、时间和精力上对学生毫无保留 |
| **思维广泛** | 惊人的记忆力，对新知识着迷，能跨领域应用 |
| **顿悟式教学** | 理念要么立刻被接受，要么永远不被接受 |
| **直觉性强** | 用生活化的比喻（桥梁承重、掷硬币）讲解投资 |
| **谦逊** | 拒绝自我标榜，警惕"自我视过高" |

## 七、关键语录汇总

### 关于投资智慧
> "要想聪明地进行证券投资，你必须事先对不同的债券和股票在不同条件下的表现有足够的知识。" — 《聪明的投资者》导言

### 关于市场波动
> "股市的行为越愚蠢，有条不紊的投资者面对的机会就越大。" — 巴菲特序言引格雷厄姆

### 关于投资者态度
> "投资者的最大问题甚至最可怕的敌人，很可能就是他们自己。" — 《聪明的投资者》

### 关于安全性原则
> "无论如何谨慎，每个投资者都免不了会犯错误。只有坚持'安全性'原则——无论一笔投资看起来多么令人神往，永远都不要支付过高的价格——你才能使犯错误的几率最小化。" — 《聪明的投资者》

### 关于"聪明"的定义
> "格雷厄姆所说的'聪明的投资者'，到底是什么意思？这种品质与智商或学术能力毫不相干。它的确切含义只是，要有耐心，要有约束，并渴望学习；此外，你还必须能够驾驭你的情绪，并能够进行自我反思。这种智慧'与其说是表现在智力方面，不如说是表现在性格方面。'" — 贾森·兹威格点评
