# 沃伦·巴菲特重大投资决策

**调研日期**：2026年4月16日

---

## 一、重大投资决策案例

### 1. 投资GEICO（政府雇员保险公司）

| 维度 | 内容 |
|------|------|
| **初次接触** | 1951年，巴菲特21岁，在格雷厄姆演讲中听到GEICO，立刻乘坐末班火车前往华盛顿调研 |
| **投资金额** | 初次投资10,282美元（当时他积蓄的约50%） |
| **逻辑** | ① 直销模式降低中间成本 ② 专为政府雇员承保（低风险群体） ③ 格雷厄姆担任董事长让他信任 |
| **关键转折** | 1970年代GEICO因扩张过度濒临破产，1976年巴菲特趁机大举买入，1980年代完全控股 |
| **结果** | GEICO成为伯克希尔核心保险业务之一，投资回报超百倍 |
| **决策模式** | "别人恐惧时贪婪"——1976年GEICO濒临破产时，巴菲特不仅没有抛售，反而大举加仓 |

**来源**：巴菲特致股东信、《滚雪球》

---

### 2. 收购伯克希尔·哈撒韦（Berkshire Hathaway）

| 维度 | 内容 |
|------|------|
| **背景** | 1962年，巴菲特发现这家濒临破产的纺织厂股价低于净资产 |
| **投资决策** | 起初买入是"雪茄烟蒂"式套利——预期清算后获利 |
| **重大错误** | 纺织厂持续亏损，但巴菲特因"沉没成本"继续投入更多资金，试图挽救 |
| **教训** | "我在伯克希尔学到的最重要的教训是：不要为错误辩解" |
| **结果** | 最终放弃纺织业务，转型为保险和投资帝国 |
| **意义** | 这个"错误"反而成就了历史上最成功的投资控股公司 |

**来源**：巴菲特1985年致股东信、《巴菲特传》

---

### 3. 投资可口可乐（Coca-Cola）

| 维度 | 内容 |
|------|------|
| **首次投资** | 1988年，投资10.2亿美元买入可口可乐 |
| **背景** | 1980年代可口可乐新CEO郭思达（Roberto Goizueta）推行改革，ROE从20%提升至31% |
| **买入逻辑** | ① 品牌护城河无可替代 ② 全球化扩张潜力 ③ 管理层优秀 ④ 现金流稳定 |
| **持有策略** | 持有至今（超过35年），期间从未大幅减持 |
| **结果** | 投资成本超过300亿美元，目前市值约2500亿美元（2025年），加上分红收益丰厚 |
| **特别之处** | 这是巴菲特"以合理价格买入优秀公司"的典型案例，而非"捡烟蒂" |

**来源**：巴菲特1988-1990年致股东信、《巴菲特之道》

---

### 4. 投资华盛顿邮报（Washington Post）

| 维度 | 内容 |
|------|------|
| **投资时间** | 1973年，投资约1,100万美元 |
| **买入逻辑** | ① 垄断性报纸的护城河 ② 股票价格远低于内在价值 ③ 凯瑟琳·格雷厄姆领导的管理层 |
| **核心指标** | 巴菲特计算其内在价值约为4-5亿美元，而当时市值仅约8,000万美元 |
| **持有期间** | 1973-1985年，约12年 |
| **卖出时机** | 1985年，分拆业务导致估值逻辑改变后卖出 |
| **结果** | 约1,100万投资变成约5,000万，年化回报超过20% |
| **经验** | "以显著低于内在价值的价格买入"是格雷厄姆式安全边际思维 |

**来源**：巴菲特1973-1985年致股东信

---

### 5. 投资比亚迪（BYD）

| 维度 | 内容 |
|------|------|
| **投资时间** | 2008年，投资约2.3亿美元买入2.25亿股比亚迪 |
| **背景** | 全球金融危机期间，市场恐慌；比亚迪是王传福领导的中国新能源汽车公司 |
| **买入逻辑** | ① 芒格推荐的"中国的丰田" ② 电池技术领先 ③ 中国政府支持新能源汽车 |
| **持有** | 持有至2022年（约14年） |
| **最高点** | 2022年比亚迪股价一度达到高点，伯克希尔持股价值约80亿美元 |
| **卖出** | 2022-2023年分批卖出，获利约30-40亿美元 |
| **特别之处** | 这是巴菲特少数几次投资中国公司；卖出时机的选择体现"能力圈"原则 |

**来源**：伯克希尔年报、比亚迪公告

## 二、投资错误与反思

### 错误1：伯克希尔·哈撒韦纺织厂（1960s-1980s）

| 维度 | 内容 |
|------|------|
| **错误本质** | 混淆"便宜"与"价值"——持续买入低于账面价值的公司，却忽视其持续亏损的业务本质 |
| **沉没成本陷阱** | 投入越多越难放弃，最终损失远超初期投资 |
| **教训** | "当发现自己身处洞中时，最重要的行动是停止挖掘" |
| **代价** | 数百万美元损失，多年精力消耗 |

### 错误2：Dexterity鞋业（1990年代）

| 维度 | 内容 |
|------|------|
| **背景** | 1990年代巴菲特收购Dexterity鞋业 |
| **错误** | 忽视全球化竞争压力，低估了低成本国家鞋业冲击 |
| **结果** | 鞋业业务最终几乎一文不值 |
| **教训** | 买入时不理解行业竞争格局的长期变化 |

### 错误3：康菲石油（ConocoPhillips）

| 维度 | 内容 |
|------|------|
| **错误时机** | 2007-2008年油价高峰时大举买入康菲石油 |
| **教训** | "在他人贪婪时恐惧"——不该在周期性行业的高点追涨 |
| **结果** | 随后油价大跌，伯克希尔承受损失 |

### 错误4：德克斯特经纪公司（Dexia）

| 维度 | 内容 |
|------|------|
| **错误** | 投资欧洲银行Dexia，忽视其资产负债表风险 |
| **教训** | 金融机构的复杂性需要特别警惕 |

## 三、决策模式分析

### 何时买？

1. **极度悲观时**：GEICO 1976年濒临破产、2008年金融危机期间投资比亚迪
2. **优质公司短暂承压时**：1988年可口可乐业绩调整期
3. **股价显著低于内在价值时**：1973年华盛顿邮报

### 何时卖？

1. **公司护城河消失时**：报纸行业因互联网衰落
2. **估值显著高估时**：比亚迪2022年高估时分批卖出
3. **逻辑改变时**：不再符合"能力圈"原则

### 为何集中？

- "把所有鸡蛋放在同一个篮子里，然后密切看好这个篮子"
- 集中投资的前提是：深入研究、极高置信度、极端机会

### 核心决策原则

| 原则 | 说明 |
|------|------|
| **护城河** | 只买具有持久竞争优势的公司 |
| **安全边际** | 价格显著低于内在价值时买入 |
| **能力圈** | 只投资自己真正理解的行业 |
| **长期持有** | 优秀公司长期持有，不轻易卖出 |
| **逆向思维** | 别人恐惧时贪婪，别人贪婪时恐惧 |
