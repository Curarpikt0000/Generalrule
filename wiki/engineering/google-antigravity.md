# Google AntiGravity

## 是什么

Google 推出的 **Agent 优先的 AI 开发平台（IDE）**。不是 SDK/库，而是一个类似 Cursor、VS Code 的集成开发环境，但核心范式是 Agent 自主工作，而非传统的手动编码。

- **发布**：2025年11月18日（公开预览）
- **官网**：[antigravity.google](https://antigravity.google)
- **官方博客**：[Introducing Google Antigravity](https://antigravity.google/blog/introducing-google-antigravity)
- **Wikipedia**：[Google Antigravity](https://en.wikipedia.org/wiki/Google_Antigravity)
- **核心模型**：Gemini 3 Pro

## 核心特性

| 特性 | 说明 |
|------|------|
| Agent 优先 | Agent 有独立工作空间，不是侧边栏聊天 |
| Artifacts | Agent 产出可验证的制品（任务清单、计划、截图、浏览器录制） |
| 浏览器控制 | Agent 可以操控浏览器 |
| 异步协作 | 多个 Agent 并行工作，开发者异步审查 |
| Manager View | 控制中心编排多个 Agent 并行工作 |
| 知识库 | Agent 保存上下文和代码片段持续学习 |
