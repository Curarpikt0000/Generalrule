## A. `Hermes Analysis ` 列(★ 这次必须改)

之前你只写了短评。现在改成:**短评 + 三条战术 合并**,放进 `Hermes Analysis ` 这个列(**注意尾随空格**)。

格式严格按下面这样写,**中文,300 字符以内**:

```
5/29 OI + 5/28 交割:【一句话定性,3~5 个最关键数字】

## B. 长文结构改造(7 个细节)

### B1. 顶部加 §0 风控仪表盘

整篇最开头,加一段**单屏可读**的仪表盘,3 行,每行一个金属。这是给"5 秒读者"看的:

```

### B2. §1~§6 每节用统一三段式

每节内部按这三段写,**结构强制一致**,方便日后做对比阅读:

```
**数据**:[原始数字,带源,可用 markdown 表]
**风控读数**:[1~2 句信号定性]
**战术含义**:[这块数据对持仓的具体启示,1 句]
```

之前你只有"数据 + 风控定性",**缺第三段"战术含义"**。补上。

每次提及大席位都带上 **3 位代码 + H/C**。这是行业标准引用方式,也方便交叉验证。

### B4. §3~§4 OI 部分加"主力 vs 远月" 拆解

之前只列总 OI + 主力月。Gemini 范例里有一条很有用的拆解:**主力月被动展期 vs 远月主动建仓**。

格式建议:
```
GC: 总 OI 348,209 (+2,504)
  ├─ 6 月主力(FND 临近)JUN26 25,382 (-6,426) ← 展期流出
  ├─ 8 月新主力 AUG26 261,561 (+7,310)  ← 接力
  └─ 远月 DEC26 32,788 (+1,818)         ← 主动建仓 ⭐
  净判定:远月 +1,818 是真增量(非展期),宏观资金加码 2027 前敞口。
```

这样一看就知道哪些是"洗手不洗脚"的展期,哪些是真增仓。

### B5. §5 CFTC 集中度加"周环比 + 历史百分位"

之前只给当周绝对值。加两个维度:

```
铂金集中度:
  G4S = 32.8% (周环比 +0.3pp, 过去 52 周 P85 → P78,趋势:略缓和)
  G8S = 50.4% (周环比 +1.1pp,P92 → P88,仍在高位)
```

P85/P78 是百分位排名——"过去 52 周 32.8% 排到第 85% 高"。**这把"高不高"变成可读的信号强度**。

### B6. §7 必须有(★)

不能因为"找起来麻烦"就 skip。**最低限度**做这两项(都是 Hermes 当前 web search 工具能查到的):

- **SGE 黄金当日溢价**:https://www.sge.com.cn/ 当日 Au(T+D) 收盘价 - COMEX 黄金 8 月合约结算价,折美元/盎司
- **SGE 白银当日溢价**:同上,Ag(T+D) - COMEX 白银 7 月合约

至于 SGE 铂金(Pt99.95)、瑞士月度出口、EFP 比例、大行观点——尽力查,**查不到的明白写"未获取"**,不要编。

### B7. 结语用"风控官三段式"

之前你的结语是直接列 3 条战术。Gemini 范例里有个更有力的结构:

```
**真相缺口(The Reality Gap)**:[纸面 vs 物理 的脱节定量描述,1~2 句]

**脱节判决(The Disconnect Verdict)**:[一句结论性定性]

**首席风控官三条战术**:
🟢 1. [行动 + 理由数字]
🟡 2. [行动 + 理由数字]
🔴 3. [行动 + 理由数字]
```

这个框架的好处:**前两段强制 Hermes 做综合,而不是 6 个维度并列后直接跳战术**。
