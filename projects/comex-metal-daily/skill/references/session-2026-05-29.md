## Web Scraping & Data API Notes (this session)

### Yahoo Finance API (preferred over scraping)
**Endpoint:** `https://query1.finance.yahoo.com/v8/finance/chart/{ticker}?range=5d&interval=1d`
- No auth required, returns clean JSON
- US futures tickers: GC=F (gold), SI=F (silver), PL=F (platinum)
- **Parse:** `['chart']['result'][0]['meta']['regularMarketPrice']` = close
- **Bulk:** also available from `['chart']['result'][0]['indicators']['adjclose'][0]['adjclose'][-1]`
- **One-liner:**
  ```bash
  curl -s 'https://query1.finance.yahoo.com/v8/finance/chart/GC=F?range=5d&interval=1d' \
    | python3 -c "import sys,json; d=json.load(sys.stdin); print(d['chart']['result'][0]['meta']['regularMarketPrice'])"
  ```
- **Note on S_fin semantics:** GC=F is continuous futures (automatic front-month roll), not a specific contract month. ~$11/below AUG26 Section62 settlement is typical contango noise. Use as ≈LBMA Fix proxy only.
