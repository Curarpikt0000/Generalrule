# v3 红绿灯聚焦版格式规范

> 适用: COMEX 贵金属深度日报 2026-05-28 起
> 设计目标: 读者 5 秒内找到当日真正关心的 1~2 个金属/维度，不被 18 处子结论平铺淹没

## 核心改造哲学

信号分级决定篇幅与位置加权:
| 灯 | 对应 | 篇幅 | 位置策略 |
|---|---|---|---|
| 🔴 Red | 必读+必行动 | 完整保留(数据+风控读数+战术含义) | 顶置,详写 |
| 🟠 Orange | 必读+密切监控 | 主要数据+一句风控读数 | 次置,半详 |
| 🟡 Yellow | 阅读+可观察 | 一段话总览+关键数字 | 后置,简写 |
| 🟢 Green | 可跳过细节 | 一行结论+必要数字 | 最后,极简 |

按这个分级,Pt 这种 🔴 的内容占整篇 50%+ 篇幅,Au/Ag 中性维度可压到 1~2 行。

## §0 风控仪表盘(页面最顶)

严格执行 18 灯表(3 金属 × 6 维度) + 综合列 + 一句话总览。

**灯判定规则(每个 cell 怎么定颜色):**

**交割(§1):**
- 🔴 单日 TOT < 100 手 且 Reg/Total > 80%(逼仓型流动性枯竭)
- 🟠 单日 TOT 100~500 手
- 🟡 TOT 异常高(>本月日均 1.5 倍)
- 🟢 TOT 在正常区间

**库存(§2):**
- 🔴 Reg/Total > 80% 且 Eligible < 1 日交割量
- 🟠 Net Change 单日 > 1% Combined 或 -1% 失血
- 🟡 Net Change 单日 0.3%~1%
- 🟢 Net Change < 0.3%

**OI 期货(§3):**
- 🔴 主力月单日 OI 变化 > ±5%
- 🟠 远月主动建仓 > 当月日均建仓量 2x
- 🟡 展期流量异常
- 🟢 OI 横盘 / 正常展期

**OI 期权(§4):**
- 🔴 C/P > 3.5x 且 CALL 单日变化 > 10%(极端看涨)或 P/C 类似(极端看跌)
- 🟠 C/P 2.5~3.5x
- 🟡 C/P 1.5~2.5x
- 🟢 C/P < 1.5x 或 OI 横盘

**CFTC(§5):**
- 🔴 G4S > 40% 或 G8S > 55%
- 🟠 G4S 30~40% 或 G8S 45~55%
- 🟡 G4S 20~30% 或 G8S 35~45%
- 🟢 G4S < 20%

**SGE/SIFO(§7):**
- 🔴 |ΔS| > 10% 且方向跟基本面共振
- 🟠 ΔS 5%~10%
- 🟡 ΔS 1%~5%
- 🟢 |ΔS| < 1%

**综合列:** 取该行最严重的灯(任一 🔴 → 综合 🔴)

### 东方库存列灯色判定规则(2026-05-31)

**Ag 行/东方库存 cell**: 取两个源(SGE+SHFE)任意一个更严重的灯
- SGE 12周累计>+100% 且持续4周以上 = 🔴
- SHFE 周环比 |ΔV| > 5% = 🟠, >10% = 🔴
- 任一源🔴→cell🔴; 任一源🟠→cell🟠; 都🟢→cell🟢

**Au 行/东方库存 cell**: 只看SHFE(SGE永久N/A)
- 周环比 |ΔV| > 5% = 🟠; >10% = 🔴

**Pt 行/东方库存 cell**: 永久N/A(置灰)

**数据源读取**: Gold DB / Silver DB filter `市场=SHFE`(或SGE) + `库存频率=每周`, 取最新行
**两源都失败**: Ag cell写"⚠数据暂缺", 绝不瞎填

### §0.5 首席风控官三条战术

位置硬性要求: 紧跟 §0 之后,用户滚动 0 次就看到。

每条格式: `[emoji灯] [金属名] [动作] | [核心数字理由,带仓位/止损建议]`

### 东方源已接入标记(2026-05-31 起每份报告必须)

1. 报告顶部新增一行: `> 本期为东方库存(SGE + SHFE)接入后第一份完整 6 维度报告。`
2. Hermes Analysis 短评开头加 `[东方源已接入]` 标签

## §1~§7 内容分层规则

### 内容按颜色压缩
- 🔴 维度/金属: 数据+风控读数(一句点名)+战术含义(怎么做)+引用源
- 🟠 维度/金属: 关键数据(最多3个数字)+一句风控读数+战术含义
- 🟡 维度/金属: 一段话总览+必要数字≤2个
- 🟢 维度/金属: 1行结论,不展开细节
  - 例: `§5 黄金 CFTC: 🟢 G4S 27.4% 中性,OI -1,651 无变化。`

### 每节标题加灯
```
### §1 实物交割流向 🔴(Pt 69手/触发逼仓预警)
### §2 库存物理流向 🔴(Pt Reg/Total 85%/Eligible 36K oz)
```

### 每个金属段落开头加灯+一句话定性
```markdown
**铂金 PL 🔴 红色**: 交割69手流动性枯竭,逼仓前置条件成熟。[数据+风控+战术]

#### 铂金(东方):
- N/A: SGE不发铂金、SHFE库存周报无铂金品种。完全靠西方+LBMA Pt CIR判断。

**黄金 GC 🟡 黄色**: 交割23,620手分歧不大方向偏多。[数据简写]

#### 黄金(东方):
- **SHFE沪金周库存**(最新周五): X.XXX吨, 环比±X.XXX吨
- 12周区间: 104~112吨, 稳定低速增长
- 风控读数: 🟢/🟠/🔴 SHFE物理层判定
- 引用源: Gold DB `市场=SHFE`

**白银 SI 🟢 绿色**: 交割1,653手温和无实质异动。[一行结论]

#### 白银(东方):
- **SGE白银周库存**(最新): X吨, 12周+XXX%累库(或排出)
- **SHFE沪银周库存**(最新周五): X.XXX吨, 环比±X.XXX吨
- SGE与SHFE方向对比: 同向/背离/分化
- 风控读数: 🟠/🔴/🟢 综合(取两源更严重灯)
- 引用源: Silver DB `市场=SGE或SHFE`
```

**东方对照篇幅纪律**: Ag双源 8~12行; Au单源 4~6行; Pt仅2行解释N/A。
**数据滞后处理**: 用最新可用周次, 加注"东方源数据滞后, 使用上周X-XX快照"。

## §9 首席风控官结语(三段式,置末)

### 段1: 真相缺口(Reality Gap)
铂金三层脱节描述 vs 黄金需求软化 vs 白银溢价常态。
格式: 每种金属一句,带数值。

### 段2: 脱节判决(Disconnect Verdict)
每种金属的判断结论 + 等待的催化剂/条件。
格式: 铂金:判断,等待催化剂; 黄金:判断,等条件; 白银:判断,叙事不变。

### 段3: 三条战术
引用回 §0.5。`三条战术: [同§0.5,见顶部]`

## 写入顺序(技术操作)

1. 建 page / 归档旧 page → create new
2. 先写 Hermes Analysis 列(短评) — 用 PATCH properties
3. 再写 blocks(正文) — 用 PATCH blocks/{id}/children
4. 分行: < 85 blocks/批,每批间隔 0.5s
